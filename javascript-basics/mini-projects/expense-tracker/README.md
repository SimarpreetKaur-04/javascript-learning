# Expense Tracker (JavaScript)

A simple console-based expense tracker built using JavaScript.

## Features
- Add expenses
- View all expenses
- Calculate total expense
- Delete an expense
- edit an expense

## Technologies
- JavaScript (Node.js)

## Learning Outcome
- Arrays & objects
- Functions
- Loops
- Clean code structure
- Array manipulation using splice
- modifying edge cases
- using Array.reduce
- refracting existing code
